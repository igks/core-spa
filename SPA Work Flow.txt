==== Membuat Komponen Baru ====
* Buat model/interface "src/app/models"
* Buat service "src/app/services"
* Buat resolver komponen di folder "src/app/resolvers"
* Buat component "src/app/main/pages/master (atau bisa membuat folder baru didalam page sesuai dengan menu).
* Tambahkan komponen ke dalam master madule ("src/app/main/pages/master/master.module")
  a. import komponen dan tambahkan dalam list/array declarations
  b. import dan tambahkan komponen service dan komponen resolver ke dalam list/array providers
  c. buat path untuk komponen pada routes
